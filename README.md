# YouTube Liquid Glass Speed Control

A modern, transparent, speed control overlay for YouTube. 

![speedControl](https://github.com/user-attachments/assets/2586a405-ffe2-4717-8d17-431ca8813fda)


## Features
* **Liquid Glass Design:** Blurs the background to blend perfectly with the video.
* **Auto-Hide:** Controls fade out automatically when the YouTube interface hides.
* **Right-Side Positioning:** unobtrusive vertical controls.
* **Precision:** Adjustable speed steps (0.25x).

## How to Install

1.  Install the [Tampermonkey](https://www.tampermonkey.net/) extension for your browser.
2.  **[CLICK HERE TO INSTALL THE SCRIPT](https://greasyfork.org/en/scripts/564770-youtube-liquid-glass-speed-control)**.
3.  Confirm the installation.

## Usage
* **+ / - Buttons:** Adjust speed by 0.25x.
* **1x / 2x Buttons:** Quick presets.
* **Scroll:** You can scroll on the overlay to change speed.
